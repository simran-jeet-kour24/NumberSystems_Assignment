# Number System Conversion Project

This project contains Python programs to convert numbers between different number systems.

## Problem 1: Decimal → Binary, Octal, Hexadecimal
- The user enters a decimal number.
- The program converts that number into binary, octal, and hexadecimal formats using Python’s built-in functions.
- Example:
  - Input: `123`
  - Output:
    - Binary   : `0b1111011`
    - Octal    : `0o173`
    - Hexa     : `0x7b`

## Problem 2: Binary/Octal/Hexadecimal → Decimal
- The user enters a number as a string (with prefixes like `0b`, `0o`, `0x`).
- Python’s `int()` function automatically detects the base and converts it to decimal.
- Example:
  - Input: `0b0110` → Output: `6`
  - Input: `0o10`   → Output: `8`
  - Input: `0x10`   → Output: `16`

---

## Summary
- **Problem 1**: Converted decimal numbers into binary, octal, and hexadecimal.
- **Problem 2**: Converted binary, octal, and hexadecimal numbers into decimal.

---

## Learning Outcomes
- Understand different number systems (decimal, binary, octal, hexadecimal).
- Learn how to use Python’s built-in functions (`bin()`, `oct()`, `hex()`, `int()`).
- Practice handling user input and type conversion in Python.
- Gain experience in writing simple, modular programs.
